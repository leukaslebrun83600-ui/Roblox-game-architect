# PRD 01 — Core Loop

> Le cœur du jeu. La boucle que le joueur répète sans s'en lasser.

---

## La boucle principale

### En une phrase
> [ACTION] → [RÉCOMPENSE] → [AMÉLIORATION] → recommencer

### Détail minute par minute

| Temps | Ce que fait le joueur | Ce qu'il ressent |
|-------|----------------------|------------------|
| 0:00 - 0:30 | | |
| 0:30 - 1:00 | | |
| 1:00 - 2:00 | | |
| 2:00 - 5:00 | | |
| 5:00 - 10:00 | | |
| 10:00+ | | |

### Boucle session (une session complète)

1. Le joueur se connecte → [que voit-il ?]
2. Il décide de [objectif de session] →
3. Il fait [actions principales] →
4. Il obtient [récompenses] →
5. Il se déconnecte satisfait parce que [raison]

### Boucle méta (progression long terme)

- **Jour 1 :** [Expérience du nouveau joueur]
- **Jour 3 :** [Ce qui a changé, ce qu'il a débloqué]
- **Semaine 1 :** [Où il en est dans la progression]
- **Mois 1 :** [Statut du joueur engagé]
- **Mois 3+ :** [Le endgame, ce qui le fait rester]

## Questions clés à valider

- [ ] Est-ce que la boucle est fun dès les 30 premières secondes ?
- [ ] Est-ce que le joueur sait toujours quoi faire ensuite ?
- [ ] Est-ce que la progression donne envie de continuer ?
- [ ] Est-ce que le rythme alterne bien entre effort et récompense ?
- [ ] Est-ce que la session a une fin naturelle (pas frustrant de quitter) ?
