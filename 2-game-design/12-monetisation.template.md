**Catégories recommandées :**
- **Cosmétique pur** (skins, titres, effets) → aucun impact gameplay
- **Confort** (vitesse, slots, x2) → plus rapide mais pas plus fort
- **Contenu** (zones, quêtes bonus) → plus de contenu à explorer

### Calcul par joueur

| Métrique | Valeur estimée |
|----------|---------------|

| % qui achète Dev Products | [Ex: 2%] |
| Dépense moyenne Dev Products/mois | [Ex: 100 Robux] |

### Projection mensuelle (à 500 DAU)

```
Game Passes : 500 × 3% × 300 Robux = 4 500 Robux/mois (nouveaux joueurs)
Dev Products : 500 × 2% × 100 Robux = 1 000 Robux/mois (récurrent)
Total : ~5 500 Robux/mois
Après commission Roblox (~75%) : ~1 375 Robux ≈ ~$5/mois

Note : Ces chiffres sont conservateurs. Les top games font beaucoup plus.
```

**Rappel taux de change Roblox :**
- 100K Robux ≈ $350 (taux DevEx)
- Roblox garde ~75% des revenus

## Éthique & Limites

### Ce qu'on NE FAIT PAS

| Pratique | Pourquoi c'est interdit |
|----------|------------------------|
| Pay-to-win (stats combat achetables) | Tue la compétition, frustre les F2P |
| Loot boxes opaques | Réglementations + mauvaise image |
| Timers de frustration abusifs | Joueur doit s'amuser, pas attendre |
| Contenu de base verrouillé | Le jeu doit être complet en F2P |
| Pression psychologique (countdown, FOMO abusif) | Public jeune, responsabilité |

### Ce qu'on FAIT

| Pratique | Pourquoi c'est bon |
|----------|-------------------|
| Cosmétiques variés | Revenus sans impact gameplay |
| Boost de confort | Le joueur choisit de gagner du temps |
| Contenu bonus | Plus de contenu pour ceux qui veulent plus |
| Transparence des taux | Afficher les probabilités de drop |
| Pity system | Garantir un drop après X tentatives |
