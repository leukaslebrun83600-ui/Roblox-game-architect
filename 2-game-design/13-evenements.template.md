# PRD 13 — Événements & Updates

> Comment le jeu reste vivant après le lancement. Cadence des mises à jour, événements saisonniers, roadmap.

## Cadence de mises à jour

| Type | Fréquence | Contenu type |
|------|-----------|-------------|
| Hotfix | Dès que nécessaire | Bug fixes, exploits |
| Update mineur | [Toutes les 1-2 semaines] | Balance, QoL, petit contenu |
| Update majeur | [Tous les 1-2 mois] | Nouvelle zone, mécaniques, items |
| Événement saisonnier | [4-6 par an] | Contenu temporaire, récompenses exclusives |

## Roadmap post-lancement

### Mois 1 (stabilisation)
- [ ] Fix bugs remontés par les joueurs
- [ ] Balance basée sur les données réelles
- [ ] Premier event mineur (bonus de lancement)

### Mois 2-3 (expansion)
- [ ] Nouvelle zone / Extension du monde
- [ ] Nouveaux items / mécaniques
- [ ] Premier événement saisonnier

### Mois 4-6 (communauté)
- [ ] Features demandées par la communauté
- [ ] Système de trading / social amélioré
- [ ] Deuxième vague de contenu

### Long terme (6-12 mois)
- [ ] Expansion majeure
- [ ] Battle Pass (si validé)
- [ ] Potentiel multi-game (universe)
